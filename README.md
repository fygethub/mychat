# websokit.io 聊天

> 环境node

## 为什么要做这个项目?

> 最近想学习一下node后台开发,相对于java后台语言来说开发比较方便快捷

## 该项目涉及的技术

> mychat: webpack +　websokit +　node->express
> 跨域请求方式

## 开发模式 开始启动

> 1. npm install 

> 2. 在一个命令窗口中启动 node index 开启node 服务

> 3. 重新打开一个窗口 webpack-dev-server 启动本地服务

> 4. 访问 http://localhost:8080/view/home.html
